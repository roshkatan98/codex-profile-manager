# codex-profile-manager v1.1.0

This release makes setup and ongoing profile management much easier.

## New

- Interactive setup wizard
- Choose how many profiles to configure
- Optional custom profile names
- Numeric names remain the default
- Remove a profile from rotation while preserving its files
- Permanently delete a verified profile directory when requested
- Uninstall the manager from the `codexpm` command
- Fully return to the original Codex setup with `--purge`

## New commands

```bash
codexpm remove <profile>
codexpm remove <profile> --delete-files
codexpm uninstall
codexpm uninstall --purge
```

## Setup

Run:

```bash
bash install.sh
```

The wizard asks how many profiles you want and whether you want to name them.

## Upgrade

```bash
git pull
bash install.sh --upgrade
codexpm doctor
```

The original Codex binary and original `~/.codex` directory remain untouched.
