# codex-profile-manager v1.1.1

This release adds a built-in updater for future versions.

## New commands

```bash
codexpm update --check
codexpm update
```

The updater:

- checks the latest stable GitHub release;
- asks for confirmation before installing;
- preserves the existing profile configuration and authentication;
- validates the downloaded archive;
- upgrades the installed manager files;
- runs `codexpm doctor` after the update.

## One-time upgrade from v1.1.0

Because v1.1.0 does not yet contain the updater, install this release once from a repository checkout:

```bash
git pull --ff-only
bash install.sh --upgrade --yes
codexpm doctor
```

After that, future stable releases can be installed with:

```bash
codexpm update
```
