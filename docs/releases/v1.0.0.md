# codex-profile-manager v1.0.0

The first public release of an unofficial profile manager for Codex CLI.

## What it does

- Manage any number of Codex authentication profiles.
- Keep each profile's authentication separate.
- Share sessions, history, configuration, and project context between profiles.
- Switch directly to a profile or rotate through profiles in order.
- Resume the latest session or start a new one.
- Keep the original Codex binary untouched.

## Main commands

```bash
codexpm list
codexpm status
codexpm use <profile>
codexpm next
codexpm add <profile>
codexpm login <profile>
codexpm logout <profile>
codexpm run
codexpm run new
codexpm doctor
```

## Platform

Linux with Bash 4+.
